Programmer avec une capteur de luminosité pour Raspberry Pi

Cette documentation est organisée en modules accessibles depuis l'onglet "Modules" de cette page.

Programmer pour la cible Raspberry Pi

Les programmes compilés pour la cible doivent être liés (linked) avec la bibliothèques liblumpiz.a (remarque : cette bibliothèque est incomplète à ce stade - vous aurez à la compléter).

Exemple avec gcc : -llumpiz

Programmer pour le simulateur Intox

Les programmes compilés pour s'exécuter avec le simulateur Intox doivent être liés (linked) avec la bibliothèque libintoxlumpiz.a (remarque : la bibliothèque libintoxlumpiz.a dépend de la bibliothèque libintox.a et de la bibliothèque système NTPL - Native Posix Thread Library).

Exemple avec gcc : -lintoxlumpiz -lintox -pthread

La compilation de ces programmes doit également définir la macro INTOX pour identifier le mode de compilation pour le simulateur.

Voir la documentation de la bibliothèque du robot que vous utilisez (par exemple, la bibliothèque libmrpiz).
